# Adidas-Sales-Dashboard

## 📌 Project Overview

This project presents an interactive Power BI dashboard created to analyze Adidas sales data. The dashboard provides insights into revenue, product performance, regional sales, and overall business trends using visually appealing charts and KPIs.

---

## 🎯 Objective

* Analyze Adidas sales performance
* Identify top-selling products and categories
* Track sales trends over time
* Understand regional and customer insights

---

## 📊 Dataset

The dataset includes key business information such as:

* Product Name
* Product Category
* Sales Amount
* Quantity Sold
* Region / Location
* Retailer
* Order Date
* Operating Profit

---

## 🛠️ Tools & Technologies

* Power BI
* Microsoft Excel / CSV
* Power Query (Data Cleaning & Transformation)

---

## ⚙️ Project Workflow

1. Data Collection
2. Data Cleaning and Preparation
3. Data Transformation using Power Query
4. Data Modeling
5. Dashboard Design
6. Data Visualization
7. Insight Generation

---

## 📈 Dashboard Features

* KPI Cards (Total Sales, Total Profit, Total Units Sold)
* Sales Trend Analysis (Monthly / Yearly)
* Top Products & Categories
* Region-wise Sales Analysis
* Retailer Performance Analysis
* Interactive Filters (Slicers)
* Drill-down Reports

---

## 🔍 Key Insights

* Identified highest revenue-generating products
* Observed peak sales periods and trends
* Compared regional sales performance
* Highlighted top-performing retailers

---

## 🚀 How to Use

1. Download the `.pbix` file
2. Open it in Power BI Desktop
3. Use filters and slicers to explore insights
